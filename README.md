# JFS_VISEM

A mini project to practice **HTML**, **CSS**, and **JavaScript** fundamentals.

## Features

- **Todo App** - Create, manage, and track your tasks
- **Course Registration** - Register for available courses with a simple form

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

1. Clone or download this project
2. Open `index.html` in your browser
3. Explore the Todo and Course Registration features

## Project Structure

```
JFS_VISEM/
├── index.html
├── styles.css
└── script.js
```

## Learning Objectives

- DOM manipulation with JavaScript
- Form handling and validation
- CSS styling and layouts
- Event handling
